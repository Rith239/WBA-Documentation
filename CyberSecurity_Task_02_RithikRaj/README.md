# Cyber Security Task 02: CIA Triad & Risk Assessment

## Objective

The purpose of this task was to understand the core principles of cybersecurity through the CIA Triad (Confidentiality, Integrity, Availability), and to learn how organizations identify and manage security risks through structured risk assessment.

## Folder Contents

| File | Description |
|---|---|
| `01_CIA_Triad_Research.md` | Definitions, importance, real-world examples, and comparison table for Confidentiality, Integrity, and Availability |
| `02_Cyber_Attack_Case_Study.md` | Case study on the WannaCry Ransomware attack — what happened, CIA impact, consequences, and prevention |
| `03_Risk_Assessment_Table.md` | Risk assessment of 5 key college assets (asset, threat, impact, risk level) |
| `04_Threat_Analysis.md` | Analysis of 5 common cybersecurity threats — description, impact, and prevention methods |
| `05_Security_Recommendations.md` | 10 security recommendations with justification for each |
| `06_Consultant_Report.md` | One-page professional report: Top 5 Risks, Top 5 Recommendations, Expected Benefits |
